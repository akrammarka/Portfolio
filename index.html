Stressy

Stressy is a stress eating app that was designed using React to help you avoid stress eating. By selecting your stress level, Stressy can help you select from healthy recipes using the Edamam API. If you don't feel like cooking, it can also show food options nearby using Google's API! This project was completed part of my final project for the Grand Circus Bootcamp.
